Evar Programming Language Documentation

Simple and Typed Scripting Language

Egyptian Variable Language

1. Overview

This language is designed to provide a simple and clear way to declare variables with explicit data types, interact with the console, and handle errors gracefully. It supports multiple data types, variable operations, and console input/output with color support the meaning of language (Evar) is Egyptian Variable.

3. Declaring Variables

Variables are declared using the Var keyword followed by the variable name, type, and optionally, the initial value.

Var <VariableName> As <DataType> = <Value>

Or without an initial value (defaults apply):

Var <VariableName> As <DataType>

Examples:

Var score As Integer = 10
Var username As String

4. Updating Variables

Assign a new value to an existing variable using the assignment operator =. The value will be converted to the variable's declared data type.

<VariableName> = <NewValue>

5. Renaming Variables

To rename a variable:

VarcName <OldName> To <NewName>

6. Deleting Variables

Remove a variable from the system:

DelVar <VariableName>

9. Error Handling

The system outputs clear error messages with color coding to indicate issues such as:

  • Unknown variable
  • Invalid data type
  • Syntax error
  • Conversion failure
